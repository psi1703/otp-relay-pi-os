# New user onboarding

## High-level process

1. Fill the yellow boxes in the **RTA_NEW User Form - AVM Amer Darwich.xlsx**.
2. Upload the filled form to the required location.
3. Send an email to Jathin.
   - Attach the filled form.
   - Add your signature or name at the end of the email.
   - Ajith sends the employee ID copy in PDF to Mustafa.
4. Jathin applies for the RTA account.
5. Jathin informs the user when the RTA account (`IITS_*USERNAME*`) is created.
6. Jathin informs Amer to request the ADM account.
7. Update the PAM onboard list and route it for approval.
8. Amer sends the PAM account creation request with the approval email attached.
9. Configure Oracle Authenticator.
10. Create requests for RDP, PAM, and SFTP.
11. Note the request ID.
12. Once VPN access is approved and closed, send the access-grant email to RTA IT Support.
13. Download and install the RTA VPN.
