# Download and install WinSCP

## Steps

1. Download and install the WinSCP tool.
2. Open the tool.
3. Use the following connection details:
   - File protocol: `SFTP`
   - Hostname: `10.11.174.40`
   - Port: `122`
   - Username: `rtadom\IITS_*USERNAME*`
   - Password: your RTA account password
