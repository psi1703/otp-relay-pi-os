New user onboarding

High-level process

  1. Fill the yellow boxes in the RTA_NEW User Form - AVM Amer Darwich.xlsx.
  2. Upload the filled form to the required location.
  3. Send an email to Jathin.
  4. Attach the filled form.
  5. Add your signature or name at the end of the email.
  6. Ajith sends the employee ID copy in PDF to Mustafa.
  7. Jathin applies for the RTA account.
  8. Jathin informs the user when the RTA account (IITS_*USERNAME*) is created.
  9. Jathin informs Amer to request the ADM account.
  10. Update the PAM onboard list and route it for approval.
  11. Amer sends the PAM account creation request with the approval email attached.
  12. Configure Oracle Authenticator.
  13. Create requests for RDP, PAM, and SFTP.
  14. Note the request ID.
  15. Once VPN access is approved and closed, send the access-grant email to RTA IT Support.
  16. Download and install the RTA VPN.
