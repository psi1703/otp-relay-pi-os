# Generated Help Output

This directory is generated by `python3 scripts/build_help_docs.py`.

Do not author content here directly.

The RTA Wizard floating guide reads:

```text
/help/wizard-guide.json
```

Screenshots used by the wizard are served from:

```text
/help/assets/
```

Authoring locations:

- Guide wording: `docs/help/*.md`
- Screenshots/images: `docs/help/assets/`
- Build logic and wizard-step mapping: `scripts/build_help_docs.py`
- Guide behavior: `frontend/app.jsx`
- Guide design: `frontend/style.css`
