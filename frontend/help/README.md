# Generated Help Output

This directory is generated by:

```bash
python3 scripts/build_help_docs.py
```

Do not author content here directly.

Source files live in:

```text
docs/help/*.md
docs/help/assets/
```

Generated outputs include:

```text
frontend/help/manifest.json
frontend/help/rendered/*.html
frontend/help/assets/*
frontend/help/wizard-guide.json
```

The RTA Wizard floating guide loads `/help/wizard-guide.json` at runtime. User-facing wizard guide text should be edited in explicit `<!-- wizard:step_id -->` blocks inside `docs/help/*.md`.
